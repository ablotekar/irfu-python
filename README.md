

# Welcome to pyrfu

PYRFU is a software based on the IRFU-MATLAB library to work with space data, particularly the Magnetospheric MultiScale (MMS) mission. 


# Instalation
pip install --index-url https://test.pypi.org/project/ --no-deps pyrf
